## 📁 TECH STACK

```
Framework  : Next.js 14 (App Router)
Language   : TypeScript (strict)
Styling    : Tailwind CSS + custom CSS
Animation  : Framer Motion
Game Canvas: Phaser 3 (untuk station grid)
```

Install semua yang dibutuhkan:

## ⚡ PHASER SCENE DETAILS

File: `lib/stationScene.ts`

```typescript
export class StationScene extends Phaser.Scene {
  // Public callbacks
  onAgentSelect?: (agent: AgentData | null) => void
  onLevelUpLabel?: (x: number, y: number, name: string, level: number) => void

  // Methods
  create()         // init rooms, agents, click handler, grid fade-in tween
  update()         // clear + draw everything tiap frame
  
  // Draw methods
  drawBackground() // fill solid bg
  drawGrid()       // dot grid 24px spacing
  drawRooms()      // 6 rooms dengan status-based border color
  drawCommLines()  // agent-to-agent flash lines  
  drawAgents()     // movement + trail + particles + body
  drawLevelUp()    // shockwave rings + burst particles + starburst
  drawScanline()   // moving CRT sweep line
  drawHUD()        // title bar overlay

  // Public API
  triggerLevelUp(agentId: string): number  // returns new level
  getAgentLevel(id: string): number
}
```

### Dynamic Import (wajib untuk SSR):
```typescript
// Di React component:
const scene = await import('@/lib/stationScene')
  .then(m => new m.StationScene())

new Phaser.Game({
  type: Phaser.AUTO,
  parent: mountRef.current,
  scene: scene,
  banner: false,
  audio: { noAudio: true }
})
```
