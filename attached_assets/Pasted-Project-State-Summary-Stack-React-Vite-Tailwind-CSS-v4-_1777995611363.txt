"Update stationScene.ts to load these assets and render the dungeon using the tileset tiles. Use the character spritesheet for agent walk animations."

Important Technical Notes
Phaser 3 canvas is inside StationCanvas.tsx which wraps stationScene.ts
stationScene.ts receives agents via scene.registry or direct data injection from the React component
Agents are passed as AgentData[] with fields: id, name, role, status, roomId
The Phaser scene re-renders when agents prop changes (via useEffect in StationCanvas.tsx)
Keep image-rendering: pixelated on the canvas for crisp pixel art scaling
Tileset must be loaded in Phaser preload() and used in create() — avoid hot-reloading conflicts
