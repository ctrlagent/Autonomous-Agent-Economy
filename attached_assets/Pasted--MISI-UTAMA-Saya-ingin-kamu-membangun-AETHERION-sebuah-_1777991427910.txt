 MISI UTAMA

Saya ingin kamu membangun **AETHERION** — sebuah AI Agent Operating System Dashboard — menggunakan **gambar-gambar UI yang saya lampirkan sebagai referensi visual 100%**.

Prioritas absolut:
- **Pixel art / 2D pixel aesthetic** persis seperti di gambar referensi
- **Warna, font, border, glow effect** harus sama identik
- **Layout dan komponen** harus mengikuti gambar secara presisi
- Tidak boleh ada deviasi desain — ini bukan interpretasi, ini adalah **reimplementasi visual yang akurat**

---

## 📁 TECH STACK

```
Framework  : Next.js 14 (App Router)
Language   : TypeScript (strict)
Styling    : Tailwind CSS + custom CSS
Animation  : Framer Motion
Game Canvas: Phaser 3 (untuk station grid)
Font       : Space Mono (monospace UI), Press Start 2P (pixel headers)
State      : React useState / useReducer
```

Install semua yang dibutuhkan:
```bash
npm install next react react-dom typescript framer-motion phaser
npm install -D tailwindcss postcss autoprefixer @types/react @types/node
```

### Typography (HARUS SAMA PERSIS DENGAN GAMBAR)
```css
/* WAJIB: Import kedua font ini */
@import url('https://fonts.googleapis.com/css2?family=Space+Mono:wght@400;700&family=Press+Start+2P&display=swap');

/* Penggunaan:
   - "AETHERION", room names, section headers → Press Start 2P
   - Stats, timestamps, badges, labels, values → Space Mono
   - Body text, task descriptions → Space Mono regular
   - TIDAK BOLEH menggunakan Inter, Roboto, atau sans-serif biasa
*/
```

### Pixel Border Style
