## Prinsip Adaptasi (PENTING)

### ✅ Yang DIAMBIL dari SAMS
- **Layout structure**: 3-column (sidebar + canvas + right panel) + bottom panel + status bar
- **Multi-view paradigm**: Spatial view, Kanban, Whiteboard, Code, CAD, Properties
- **Command palette pattern** (CMD+K)
- **Explorer tree** di sidebar
- **Minimap** di kanan atas
- **Bottom multi-tab panel** (terminal, events, output)
- **Status bar** di paling bawah
- **AI Assistant** sebagai right panel context-aware

### ❌ Yang TIDAK DIAMBIL (di-replace dengan identity CTRL)
- **Visual style**: SAMS modern 3D smooth → CTRL **pixel art 2.5D**
- **Color**: SAMS soft pastels → CTRL **dark navy + neon glow**
- **Typography**: SAMS Inter → CTRL **Press Start 2P + Space Mono**
- **Karakter**: SAMS 3D rounded mascots → CTRL **8×17 pixel agents**
- **Metafora**: SAMS office → CTRL **space station**
- **Terminologi**: SAMS spatial/primitive → CTRL **station/room/mission/airlock**
- **Theme**: SAMS terang → CTRL **dark cyberpunk command center**

Lihat Section 1 di `../REPLIT-UPGRADE-TASKS.md` untuk daftar lengkap.
